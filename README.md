# Opened_Seri_Project
이 프로젝트는 SERISES _DD에서 만든 개인 프로젝트로 제출하기 위하여 개인 API코드를 제외 한 나머지 코드들을 올림 만약 실행 해보고 싶다면
+API_KEY 이 부분에 본인의 YOUTUDE API V3를 받아 넣고 실행을 하면 된다.

그외 설명은 따로 내 TISTORY 주소를 첨부 합니다.
나중에 코드가 바뀌거나 업데이트 되면 수정될 가능성이 있습니다.
